Repositório dedicado à centralização de todas as atividades, desafios de código e projetos desenvolvidos por mim durante o Bootcamp de **Desenvolvedor Java Fullstack** na **Generation Brasil** (Turma 85). 

O objetivo deste espaço é registrar toda a minha jornada de aprendizado, evolução na lógica de programação e aplicação prática dos conceitos do ecossistema Java.


---

## 🛠️ Tecnologias & Conceitos em Evolução

À medida que o bootcamp avança, este repositório é atualizado com exercícios que cobrem:

* **Linguagem:** Java 17+ ☕
* **Paradigma:** Programação Orientada a Objetos (POO) 🧩
* **Estrutura de Dados:** Collections (`List`, `ArrayList`, `Set`, `Optional`) 🗃️
* **Boas Práticas:** Tratamento de exceções, modularização e arquitetura limpa 🏛️
* **Versionamento:** Git e GitHub para controle de versão 💻
